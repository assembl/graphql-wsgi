# wsgi_graphql

Port of [express-graphql](https://github.com/graphql/express-graphql)
to Python and WSGI. Uses [graphqllib](https://github.com/dittos/graphqllib).

## Project Status

**This project is highly experimental. Please do not use this in
  production yet.**
